# CursorFollow — 800×800 для OBS

Окно **800×800** показывает область монитора вокруг курсора. OBS: **Захват окна** → **`CursorFollow — OBS Capture`**.

https://github.com/user-attachments/assets/d08905e7-6417-4dbe-b224-f9d4e25d2ed4

## Запуск

```powershell
cd c:\Users\ords\Desktop\projs\obsplugins\cursor_follow_app
pip install -r requirements.txt
py -3 main.py
```

Или двойной клик: `start.bat` (пресет **stream**, **60 FPS**).

Параметры: `--preset stream|snappy|smooth` `--fps 60` `--dead-zone 80` `--lerp 0.12`

Кадр обновляется **всегда 60 раз/с** (анимации, скролл, печать). **Курсор рисуется в окне** — в OBS «Захватывать курсор» держите **выкл**.

## OBS — захват окна

1. **Сначала запустите** `py -3 main.py` — окно должно быть открыто (не свёрнуто).
2. OBS → источник **Захват окна**.
3. В списке **Окно** выберите:  
   **`[python.exe]: CursorFollow — OBS Capture`**  
   (или `py.exe` — смотрите по заголовку **CursorFollow — OBS Capture**).
4. Настройки источника:
   - **Захватывать курсор** — **выкл** (курсор уже рисуется приложением).
   - **Область клиента** — **вкл** (ровно 800×800).
   - Метод захвата Windows 10/11 — по умолчанию; если артефакты — попробуйте другой.
5. На сцене **1080×1920** (Shorts):
   - вебка **1080×607** сверху;
   - захват окна масштаб **1080×1080** снизу (апскейл с 800 — чётко для Shorts);
   - crop в OBS — **0**, фильтр **Ланцош** при масштабе (если есть).

## Пресеты (вайбкодинг)

| Пресет | Мёртвая зона | Lerp | Когда |
|--------|--------------|------|--------|
| **stream** (default) | 80 px | 0.12 | Стрим, код, плавно |
| snappy | 55 px | 0.20 | Быстрее реакция |
| smooth | 110 px | 0.08 | Максимально спокойно |

## Схема

```
Монитор → mss 800×800 → [CursorFollow — OBS Capture] → OBS Window Capture → Shorts
```

## Свёрнутое окно

- Свернуть можно (**кнопка «—»** в заголовке или панель задач).
- Пока окно свёрнуто, захват **пауза** (меньше нагрузка на CPU).
- **На стриме** разверните окно — иначе OBS «Захват окна» часто даёт чёрный кадр.
- Удобно: свернуть, когда не стримите; перед эфиром — развернуть.

## Важно

- Перетащите окно на 2-й монитор, если мешает — в OBS картинка та же.
- Захват с **основного монитора**.
